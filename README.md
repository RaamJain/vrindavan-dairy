# Vrindavan Dairy

Version 1.0.0

Desktop dairy management system.

## Features

- Customer Management
- Product Management
- Daily Milk Entry
- Extras Management
- Monthly Billing
- Payments
- Reports
- Single Bill Printing
- Bulk Bill Printing
- Automatic Excel Generation
- Automatic Database Creation
- Automatic Backups
- Desktop Application (macOS / Windows)

## Technology

Backend

- Python
- FastAPI
- SQLAlchemy
- SQLite
- OpenPyXL

Frontend

- React
- TypeScript
- TailwindCSS
- Vite

Desktop

- Electron

## Storage

Application data is stored in:

macOS

~/Library/Application Support/Vrindavan Dairy/

Windows

%APPDATA%\Vrindavan Dairy\

## Backup

Automatic backups are created every startup.
